# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-12
### Added
- **Syntax Standardization**: Brace-based C-style syntax finalized for all language constructs.
- **Peephole Optimizer**: Constant folding and redundant code removal.
- **Weak Pointers**: `weak<T>` support for cycle breaking.
- **vyx Package Manager**: GitHub-based decentralized package manager.
- **CI/CD**: GitHub Actions for multi-platform builds.
- **LSP**: Inline diagnostics, Go-to-Definition, and Hover support.
- **Professional Setup**: Issue templates, PR templates, and community docs.

### Changed
- Improved IR verification and VM stability.
- Updated VS Code extension with enhanced language support.

## [0.5.0] - Previous
- Initial Beta release with Closures and Generics.
