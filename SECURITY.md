# Security Policy

## Supported Versions
Only the latest version of VYRONIX is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability within VYRONIX, please send an e-mail to `indomoro94@gmail.com`. All security vulnerabilities will be promptly addressed.
