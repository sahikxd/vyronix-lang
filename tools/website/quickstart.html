🚀 Quickstart Guide

Get up and running with VYRONIX in less than 5 minutes.

1. Installation

Download the latest vyronix and vyx binaries from the Releases page. Add them to your system PATH.

2. Your First Program

Create a file named hello.vx:

// hello.vx
fn main() { 
    let name = "Developer"; 
    io.print("Hello, " + name + "! Welcome to VYRONIX."); 
} 

main();

3. Running the Code

Use the run command to compile and execute instantly:

vyronix run hello.vx

4. Compiling to Bytecode

For distribution, compile to the optimized .vyb format:

vyronix build hello.vx -o hello.vyb 
vyronixvm hello.vyb

5. Using the Package Manager

Initialize a project and install dependencies:

vyx init 
vyx install http
