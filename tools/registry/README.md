# 📦 VYRONIX Package Registry

This is the official decentralized package registry for the **VYRONIX** programming language.

## 🚀 Usage

To install a package from this registry, use the `vyx` package manager:

```bash
vyx install <package-name>
```

Example:
```bash
vyx install http
```

## 🛠 Contributing

To add your own package to this registry, please submit a Pull Request updating the `packages.json` file.
